Nom du Projet 
   |- Dual Uno 
Description 
   |- Un Uno qui ce joue sur deux postes différents via à un serveur sur le réseau local 
Décrire les fonctionnalités
   |- Toutes les régle de base du uno (https://www.jeuxuno.com/regles-officielles)
Scénario d'utilisation
   |- Chaque joueur reçoit 7 cartes choisies aléatoirement dans la pioche, toutes couleurs et valeurs confondues. Le joueur pose une carte si elle correspond en couleur ou en valeur à la carte sur le dessus du tas de défausse, sinon il pioche une carte, si la carte piochée est jouable, il peut la poser tout de suite. Si la carte du dessus de la défausse est Rouge 5, le joueur peut jouer toute carte Rouge ou toute carte 5 (ou une carte spéciale compatible).
